# 🛠️ COMANDOS ÚTILES - Testing y Debugging WhatsApp\n\n**Documento rápido:** Comandos SQL, Node.js y terminal para testing\n\n---\n\n## 📊 COMANDOS SQL (Supabase)\n\n### Ver todas las plantillas\n\n```sql\nSELECT id, nombre, tipo, activa, created_at \nFROM public.plantillas_whatsapp \nORDER BY created_at DESC;\n```\n\n### Contar plantillas activas\n\n```sql\nSELECT COUNT(*) as total_plantillas \nFROM public.plantillas_whatsapp \nWHERE activa = true;\n-- Esperado: 10\n```\n\n### Ver mensajes enviados (últimas 24h)\n\n```sql\nSELECT \n  COUNT(*) as total,\n  COUNT(CASE WHEN estado='enviado' THEN 1 END) as exitosos,\n  COUNT(CASE WHEN estado='fallido' THEN 1 END) as fallidos\nFROM public.whatsapp_mensajes\nWHERE created_at > NOW() - INTERVAL '24 hours';\n```\n\n### Ver mensajes por tipo\n\n```sql\nSELECT \n  tipo,\n  COUNT(*) as total,\n  COUNT(CASE WHEN estado='enviado' THEN 1 END) as exitosos,\n  COUNT(CASE WHEN estado='fallido' THEN 1 END) as fallidos,\n  ROUND(100.0 * COUNT(CASE WHEN estado='enviado' THEN 1 END) / COUNT(*), 2) as porcentaje_exito\nFROM public.whatsapp_mensajes\nGROUP BY tipo\nORDER BY total DESC;\n```\n\n### Ver últimos 10 mensajes\n\n```sql\nSELECT \n  telefono,\n  tipo,\n  estado,\n  mensaje_texto,\n  created_at\nFROM public.whatsapp_mensajes\nORDER BY created_at DESC\nLIMIT 10;\n```\n\n### Ver mensajes fallidos\n\n```sql\nSELECT \n  telefono,\n  tipo,\n  estado,\n  mensaje_texto,\n  error,\n  created_at\nFROM public.whatsapp_mensajes\nWHERE estado = 'fallido'\nORDER BY created_at DESC\nLIMIT 20;\n```\n\n### Ver teléfonos sin respuesta (24h+)\n\n```sql\nSELECT \n  telefono,\n  ultimo_mensaje_out,\n  ultima_interaccion,\n  total_mensajes_in,\n  total_mensajes_out,\n  NOW() - ultima_interaccion as tiempo_sin_respuesta\nFROM public.whatsapp_conversaciones\nWHERE estado = 'activa'\n  AND respuesta_recibida IS NULL\n  AND ultima_interaccion < NOW() - INTERVAL '24 hours'\nORDER BY ultima_interaccion ASC;\n```\n\n### Estadísticas diarias\n\n```sql\nSELECT * FROM public.vw_whatsapp_stats_diarias\nORDER BY fecha DESC\nLIMIT 7;\n```\n\n### Usuarios más activos\n\n```sql\nSELECT * FROM public.vw_whatsapp_usuarios_activos\nLIMIT 10;\n```\n\n### Leads más comprometidos\n\n```sql\nSELECT * FROM public.vw_whatsapp_leads_activos\nLIMIT 10;\n```\n\n### Buscar mensajes de un usuario específico\n\n```sql\nSELECT * \nFROM public.whatsapp_mensajes\nWHERE usuario_id = 'uuid-del-usuario'\nORDER BY created_at DESC;\n```\n\n### Buscar mensajes por teléfono\n\n```sql\nSELECT * \nFROM public.whatsapp_mensajes\nWHERE telefono = '573001234567'\nORDER BY created_at DESC;\n```\n\n### Limpiar mensajes de test (CUIDADO)\n\n```sql\n-- Solo si necesitas eliminar todos (usar con cuidado)\nDELETE FROM public.whatsapp_mensajes \nWHERE usuario_id = 'test-user-id';\n\n-- Verificar cuántos se van a eliminar primero:\nSELECT COUNT(*) \nFROM public.whatsapp_mensajes \nWHERE usuario_id = 'test-user-id';\n```\n\n### Ver tamaño de la tabla\n\n```sql\nSELECT \n  table_name,\n  pg_size_pretty(pg_total_relation_size(schemaname||'.'||table_name)) as size\nFROM pg_tables\nWHERE table_name LIKE 'whatsapp%'\n  AND schemaname = 'public';\n```\n\n### Ver crecimiento diario\n\n```sql\nSELECT \n  DATE(created_at) as fecha,\n  COUNT(*) as mensajes_ese_dia,\n  SUM(COUNT(*)) OVER (ORDER BY DATE(created_at)) as total_acumulado\nFROM public.whatsapp_mensajes\nGROUP BY DATE(created_at)\nORDER BY fecha DESC;\n```\n\n---\n\n## 🧪 COMANDOS NODE.JS / JAVASCRIPT\n\n### Importar el módulo (en console del navegador o script)\n\n```javascript\nimport { \n
