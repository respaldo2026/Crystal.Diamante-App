# 🤖 Configuración Make (Zapier) para WhatsApp\n\n**Propósito:** Integrar Make con la Academia para automatizar:\n- 📱 Responder publicidad de redes\n- 📩 Enviar información de cursos\n- 🎯 Gestionar leads automáticamente\n\n---\n\n## 1️⃣ Escenario Base: Responder Lead Interesado\n\n### Trigger: Lead llena formulario en redes/web\n\n**Módulos en Make:**\n```\n1. TRIGGER: Formulario web (Google Forms o custom)\n   ├─ Obtener: nombre, telefono, curso_interes, ciudad\n\n2. HTTP REQUEST: Enviar mensaje WhatsApp\n   ├─ URL: https://tu-dominio.com/api/whatsapp/send\n   ├─ Method: POST\n   ├─ Headers:\n   │  ├─ x-api-key: {{WHATSAPP_API_KEY}}\n   │  └─ Content-Type: application/json\n   ├─ Body:\n   │  {\n   │    \"phone\": \"{{telefono}}\",\n   │    \"type\": \"text\",\n   │    \"message\": \"Hola {{nombre}},\\n\\n¿Te interesa {{curso_interes}}?\\n\\nResponde este mensaje y te paso toda la info.\"\n   │  }\n\n3. SUPABASE: Crear registro de lead\n   ├─ Tabla: leads\n   ├─ Datos: nombre, telefono, curso_interes, fuente, created_at\n\n4. FILTRO: ¿Respondió en 24h?\n   └─ Si no, pasar a escalada\n\n5. HTTP REQUEST: Enviar seguimiento\n   ├─ Esperar 2 días\n   └─ Enviar mensaje de descuento\n```\n\n---\n\n## 2️⃣ Flujo: Enviar Información de Curso\n\n### Trigger: Lead responde \"Cuéntame más\"\n\n**Módulos:**\n```\n1. WEBHOOK: Recibir mensaje WhatsApp\n   ├─ Detectar palabras clave: \"info\", \"cuéntame\", \"más\", \"detalles\"\n   ├─ Usuario: {{telefono_usuario}}\n\n2. BUSCAR en BD: ¿Existe este lead?\n   ├─ Tabla: whatsapp_conversaciones\n   ├─ Buscar por: telefono\n\n3. SI NO EXISTE:\n   ├─ Crear en whatsapp_conversaciones\n   ├─ Crear en leads (si es nuevo)\n\n4. HTTP REQUEST: Enviar información del curso\n   ├─ Llamar a función: enviarInformacionCurso\n   ├─ URL: https://tu-dominio.com/api/whatsapp/send\n   ├─ Body:\n   │  {\n   │    \"phone\": \"{{telefono}}\",\n   │    \"type\": \"text\",\n   │    \"message\": \"Aquí está la info:\\n\\nCurso: {{curso}}\\nDuración: {{duracion}}\\nCosto: {{costo}}\\nInscríbete: {{link}}\"\n   │  }\n\n5. LOG: Guardar interacción\n   ├─ Tabla: whatsapp_conversaciones\n   ├─ Update: ultimo_mensaje_out, total_mensajes_out\n```\n\n---\n\n## 3️⃣ Flujo: Lead Interesado → Crear en App\n\n### Trigger: Lead dice \"Sí, me interesa\"\n\n**Módulos:**\n```\n1. WEBHOOK: Recibir mensaje\n   ├─ Detectar: \"sí\", \"me interesa\", \"quiero\", \"inscribir\"\n\n2. SUPABASE: Actualizar lead\n   ├─ Tabla: leads\n   ├─ Campos: estado='calificado', fecha_respuesta\n\n3. HTTP REQUEST: Guardar conversación\n   ├─ URL: https://tu-dominio.com/api/leads/guardar-interaccion\n   ├─ Body: { telefono, mensaje, fecha }\n\n4. HTTP REQUEST: Enviar link de inscripción\n   ├─ URL: https://tu-dominio.com/api/whatsapp/send\n   ├─ Message: \"¡Perfecto! Aquí está tu link de inscripción: {{link_inscripcion}}\"\n\n5. NOTIFICACIÓN: Avisar a admin\n   ├─ Email a: admin@academia.com\n   ├─ Asunto: \"Nuevo lead calificado - {{nombre}}\"\n```\n\n---\n\n## 4️⃣ Flujo: Recordatorios Automáticos\n\n### Recordatorio de Pago (Diario)\n\n**Trigger:** Cada día a las 10 AM\n\n```\n1. SCHEDULE: 10:00 AM Colombia\n\n2. SUPABASE: Obtener pagos vencidos\n   ├─ Query: pagos WHERE estado='pendiente' AND fecha_vencimiento < TODAY+5d\n\n3. LOOP: Para cada pago\n   ├─ HTTP REQUEST: Enviar recordatorio\n   ├─ URL: https://tu-dominio.com/api/whatsapp/send\n   ├─ Body:\n   │  {\n   │    \"phone\": \"{{telefono_usuario}}\",\n   │    \"type\": \"text\",\n   │    \"message\": \"Recordatorio: Tu pago de ${{monto}} vence el {{fecha_vencimiento}}\\nResponde si ya lo hiciste\"\n   │  }\n\n4. LOG: Guardar en whatsapp_mensajes (automático por API)\n```\n\n---\n\n## 5️⃣ Flujo: Seguimiento de Leads No Respondidos\n\n### Trigger: Cada 2 días, si no hay respuesta\n\n```\n1. SCHEDULE: Cada 2 días\n\n2. SUPABASE: Obtener leads sin respuesta\n   ├─ Query: leads WHERE estado='contactado' AND updated_at < 48h ago\n\n3. LOOP: Para cada lead\n   ├─ HTTP REQUEST: Enviar mensaje de descuento\n   ├─ Message: \"No te pierdas! Descuento 20% válido hoy 🎁 {{link}}\"\n\n4. ACTUALIZAR: Cambiar estado a 'seguimiento'\n\n5. SI RESPONDE:\n   ├─ Cambiar estado a 'interesado'\n   └─ Notificar a admin\n```\n\n---\n\n## 6️⃣ Webhook Entrante: Recibir Mensajes\n\n### Configurar webhook en Meta/WhatsApp\n\n**URL:** `https://tu-dominio.com/api/whatsapp/webhook`\n\n**Métodos en Make:**\n```\n1. WEBHOOK MAKE: Recibir POST de WhatsApp\n   ├─ Campo: text.body (contenido del mensaje)\n   ├─ Campo: from (teléfono del usuario)\n   ├─ Campo: timestamp (cuándo llegó)\n\n2. PARSE JSON: Extraer datos\n   ├─ Teléfono: {{webhook.messages[0].from}}\n   ├─ Mensaje: {{webhook.messages[0].text.body}}\n\n3. CONDICIONES:\n   ├─ ¿Es keyword? \"info\", \"precio\", \"inscribir\"\n   ├─ ¿Es lead existente?\n   ├─ ¿Está en ventana de 24h?\n\n4. ACCIONES SEGÚN PALABRA CLAVE:\n   ├─ \"info\" → Enviar información\n   ├─ \"precio\" → Enviar costos\n   ├─ \"inscribir\" → Enviar link\n   ├─ \"horario\" → Enviar horarios\n   ├─ Otros → Respuesta genérica\n\n5. GUARDAR CONVERSACIÓN:\n   ├─ Update whatsapp_conversaciones\n   ├─ Fields: ultimo_mensaje_in, total_mensajes_in\n```\n\n---\n\n## 🔑 Variables de Entorno para Make\n\nGuardar en Make como \"Variables de texto\"\n\n```\nWHATSAPP_API_URL=https://tu-dominio.com\nWHATSAPP_API_KEY={{tu-api-key-aqui}}\nSUPABASE_URL={{tu-supabase-url}}\nSUPABASE_KEY={{tu-supabase-key}}\nWHATSAPP_NUMBER={{tu-numero-de-whatsapp}}\n```\n\n---\n\n##
