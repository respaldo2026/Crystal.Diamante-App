Inscripción → Confirmación\n\n```\nUsuario llena formulario\n         ↓\n    Validar datos\n         ↓\n  INSERT inscripción (BD)\n         ↓\n  enviarConfirmacionInscripcion()\n         ↓\n  Obtener plantilla: \"inscripcion_confirmada\"\n         ↓\n  Reemplazar variables:\n  - {{nombre}} → Juan\n  - {{nombre_curso}} → Python\n  - {{horario}} → 2:00 PM\n  - {{mensualidad}} → $250.000\n         ↓\n  POST /api/whatsapp/send\n         ↓\n  ✅ Mensaje en WhatsApp de Juan\n  ✅ Log en whatsapp_mensajes\n  ✅ Redirigir a dashboard\n```\n\n### 2. Pago → Confirmación\n\n```\nProceso de pago completado\n         ↓\n  UPDATE pagos SET estado='pagado'\n         ↓\n  enviarConfirmacionPago()\n         ↓\n  Obtener plantilla: \"pago_recibido\"\n         ↓\n  Reemplazar variables:\n  - {{nombre}} → Juan\n  - {{monto}} → $250.000\n  - {{referencia_pago}} → REF123ABC\n         ↓\n  POST /api/whatsapp/send\n         ↓\n  ✅ Confirmación en WhatsApp\n  ✅ Recibo guardado\n```\n\n### 3. Info de Curso\n\n```\nUsuario en página de curso\n         ↓\n  Click \"📱 Enviarme por WhatsApp\"\n         ↓\n  enviarInformacionCurso()\n         ↓\n  Obtener plantilla: \"informacion_curso\"\n         ↓\n  Reemplazar 15 variables:\n  - nombre, curso, duración\n  - horario, costo, cupos\n  - link de inscripción, etc\n         ↓\n  POST /api/whatsapp/send\n         ↓\n  ✅ Info en WhatsApp\n  ✅ Log guardado\n  ✅ Usuario puede hacer click en link\n```\n\n### 4. Recordatorio de Pago (Cron)\n\n```\nCada día 10:00 AM\n         ↓\n  GET /api/cron/recordatorios-pago\n         ↓\n  Query: SELECT pagos WHERE estado='pendiente' \n         AND fecha_vencimiento BETWEEN hoy+3d AND hoy+5d\n         ↓\n  LOOP: Para cada pago\n    ↓\n    enviarRecordatorPago(usuario)\n    ↓\n    Plantilla: \"recordatorio_pago\"\n    ↓\n    POST /api/whatsapp/send\n    ↓\n  ✅ Recordatorio en WhatsApp\n         ↓\n  INSERT whatsapp_mensajes (100 registros)\n         ↓\n  Log: \"Enviados 100 recordatorios\"\n```\n\n### 5. Recordatorio de Clase (Cron)\n\n```\nCada día 9:00 AM\n         ↓\n  GET /api/cron/recordatorios-clase\n         ↓\n  Query: SELECT clases WHERE fecha_hora \n         BETWEEN ahora-30m AND ahora+1h\n         ↓\n  LOOP: Para cada clase\n    ↓\n    LOOP: Para cada estudiante inscrito\n      ↓\n      enviarRecordatorioClase()\n      ↓\n      Plantilla: \"recordatorio_clase\"\n      ↓\n      POST /api/whatsapp/send\n      ↓\n  ✅ Recordatorio 1 hora antes\n         ↓\n  Sistema automático sin intervención\n```\n\n---\n\n## 🗄️ ESQUEMA DE
